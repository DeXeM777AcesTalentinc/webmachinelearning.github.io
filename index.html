The mission of the W3C Web Machine Learning Working Group is to enable efficient machine learning inference in the browser. This Working Group collaborates with its sister Community Group that incubates related specifications to be adopted by the Working Group for standardization. To learn more on how we embarked on this journey and what's ahead, please see the W3C blog post.

Interested in helping with this mission? Join us! Here's how:

  1. Create a W3C Account
  2. Join the Community Group, and if representing a W3C Member, also join the Working Group
  3. That's all! You're all set to contribute on GitHub and join meetings
